    About Koi Shiba

    KoiShiba Inspired By Shiba Inu.

    KoiShiba which works on Binance Smart Chain, the latest meme cryptocurrency inspired by Shiba Inu. More specifically, it is a hyper-deflationary subcoin with a smart sharing system built into the ecosystem, so each investor gets more KoiShiba added to their wallet. This means that each owner gets a 4% commission fee for every transaction that takes place.
